Gulp is awesome

Heck yeah, let's bundle and minify stuff

  1. To get started with Gulp, I need to create a gulpfile.js
  2. In our gulpfile.js, we need to require a few libraries:
  3. gulp, gulp-uglify, gulp-concat, and gulp-watch.
  4. Gulp is for running tasks
  5. Uglify is for minification/obfuscation
  6. Concat is for concatenation
  7. Watch will listen for changes to your code and auto-run the tasks when changes are made
  8. Now, we need to install our libraries... so for each of them install with --save-dev. Example: npm install gulp-watch --save-dev
  9. We then need to define settings. Check the gulpfile for an example.
  10. After that, I create my first gulp.task named default.
  11. Gulp requires that there is a default task.
